# ThinkJS 学习

ThinkJS 是一款面向未来的 Node.js 开发框架，整合了很多最佳实践，让企业级开发变得更加简 单、高效。从 3.0 开始，ThinkJS 基于 koa 2.x，完全兼容 koa 里的 middleware 等插件。

同时，ThinkJS 支持 Extend 和 Adapter 等方式，方便扩展框架里的各种功能。

  • 官方地址：前往查看 (opens new window)

# 特性

  • 支持 Middleware、Extend、Adapter 等扩展方式
  • 基于 Koa 2.x 开发
  • 性能优异，单元测试覆盖程度高
  • 内置自动编译、自动更新机制，方便开发环境快速开发
  • 直接使用 async/await解决异步问题，不支持 */yield

# 使用

# 安装脚手架
npm install -g think-cli

# 创建项目
thinkjs new demo

# 进入项目
cd demo

# 安装依赖
npm install

# 运行项目
npm start
