# nvm 学习与使用

nvm 是 Node.js 的版本管理工具

Github地址：前往查看 (opens new window)

# 查看本地安装的所有版本；有可选参数available，显示所有可下载的版本。
nvm list [available]

# 安装，命令中的版本号可自定义，具体参考命令1查询出来的列表
nvm install 11.13.0

# 使用特定版本
nvm use 11.13.0

# 卸载
nvm uninstall 11.13.0
